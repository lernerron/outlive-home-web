## Affected Pages/Routes
- `/` — Homepage promoted from HomePremium. Verify all sections render, animations work, CTAs open lead form.
- `/services/bathroom-safety` — New landing page. Verify all 9 sections, pricing from services.json, FAQ accordion, phone tap-to-call, qualified lead form.
- `/partners` — New partner page. Verify partner form submits with leadType: "partner", OutliveHome.org link, footer nav link.
- `/testing` — Should 301 redirect to `/`
- `/demo` — Should 301 redirect to `/`

## Key Interactions to Verify
- Lead form modal opens from all CTA buttons on homepage and bathroom-safety page
- Lead form submits with new qualification fields (serviceType, urgency, relationship)
- Lead form submits WITHOUT qualification fields (backward compatibility)
- Partner form submits without zipCode (conditional validation)
- FAQ accordion opens one section, closes others
- Phone number is a clickable tel: link
- Navigation includes "Bathroom Safety" in header and "Partners" in footer

## Edge Cases
- Form submission with empty optional fields (serviceType, urgency, relationship all blank)
- Partner form submission without zipCode — API must accept
- Consumer form submission without zipCode — API must reject
- services.json with null startingPrice — should show "Get a Free Quote" instead of price
- Very long organization name in partner form
- Mobile viewport: all layouts single-column, touch targets 48px+

## Critical Paths
- Consumer lead: Click CTA → Fill form with qualification → Submit → API validates → Webhook fires → Google Sheets receives data with new fields
- Partner lead: Click "Become a Partner" → Fill partner form → Submit → API validates (no zipCode required) → Webhook fires with leadType: "partner"
- Redirect: Visit /testing → 301 to / → Homepage loads
